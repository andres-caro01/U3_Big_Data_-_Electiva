Conectores para Spark - Databricks

Ofrecen el poder ejecutar consultas directamente sobre un clúster de Spark (Databricks, 2025).

Conectores para Cloud Data Warehouses

Se conectan a servicios como Amazon Redshift, Google BigQuery o Azure Synapse Analytics, delegando el procesamiento de las consultas al potente motor del data warehouse.

Modos de conexión

Ofrecen diferentes modos de conexión para equilibrar el rendimiento y la actualidad de los datos.

El modo Import copia los datos en el motor en memoria para rendimiento rápido. El modo DirectQuery garantiza datos actualizados enviando consultas en tiempo real.
