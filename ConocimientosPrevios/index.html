REQUISITOS

Conocimientos previos

  • Conceptos básicos de Sistemas Distribuidos: tener conocimiento básico de almacenamiento distribuido (HDFS), procesamiento paralelo (MapReduce y Spark).
  • Programación en lenguajes como Python o Scala: conocer y programar en alguno de los anteriores lenguajes, manejo de sus estructuras de datos, trabajar con programación funcional y uso de funciones, tanto de usuario como map, filter y reduce entre otras.
  • Lenguaje de Consulta SQL: tener conocimiento en manejo y sintaxis de consultas en SQL, como sentencias: SELECT- FROM – WHERE - GROUP BY y JOINs.
  • Conceptos de Bases de Datos: conocimiento de los modelos de datos relacionales y no relacionales, y sus casos de uso.
  • Estadística descriptiva básica: familiaridad con conceptos como media, mediana, desviación estándar y la interpretación de histogramas.

OBJETIVOS

Competencias

  • Comparar y contrastar las arquitecturas y casos de uso de las principales herramientas de ingesta de datos (Sqoop, Flume, Kafka), seleccionando la más adecuada para escenarios de transferencia en batch y en streaming.
  • Desarrollar y ejecutar consultas analíticas complejas sobre DataFrames de Spark utilizando Spark SQL, aplicando transformaciones, agregaciones y funciones de ventana para realizar análisis exploratorios de datos (EDA) a gran escala.
  • Aplicar algoritmos fundamentales de aprendizaje automático no supervisado (k-means) y supervisado (clasificación básica) utilizando la API de pipelines de Spark MLlib para construir, entrenar y evaluar modelos predictivos introductorios.
  • Evaluar y seleccionar servicios gestionados de Big Data de los principales proveedores de nube (AWS, Azure, GCP), diseñando una arquitectura de alto nivel que integre componentes de ingesta, almacenamiento, procesamiento y análisis, y justificando las decisiones en función de los requisitos técnicos y las consideraciones de costos.
  • Conectar herramientas de visualización (como Power BI o Tableau) a fuentes de Big Data y aplicar principios de diseño visual para comunicar eficazmente los insights derivados del análisis de grandes volúmenes de datos.
