A
Amazon Web Services (AWS)
Como pionero en la computación en la nube, AWS tiene el ecosistema de servicios más maduro y extenso. EMR es altamente configurable y popular para cargas de trabajo de Spark. Redshift es un Data Warehouse MPP maduro, y Kinesis es su plataforma para el streaming de datos en tiempo real (Amazon Web Services, 2026).
M
Microsoft Azure
Es una opción fuerte para empresas que ya están invertidas en el ecosistema de Microsoft. El sistema HDInsight ofrece clústeres gestionados para la mayoría de frameworks de tipo código abierto. Azure Synapse Analytics es la plataforma de análisis integrada, la cual combina el data warehousing, integración de datos y todo el análisis de Big Data (Microsoft, s.f.).
G
Google Cloud Platform – GCP
Esta plataforma se caracteriza por su fortaleza en machine learning, análisis de datos y redes. BigQuery es un servicio Data Warehouse serverless (Bodega de Datos sin servidor), muy potente y fácil de usar. Esta plataforma separa en forma total el almacenamiento del cómputo. Dataflow, como otro servicio, ofrece un modelo de programación unificado, Apache Beam, que ofrece tanto procesamiento en streaming como en batch (Google Cloud, S.f.).
