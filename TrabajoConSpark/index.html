> FEATURE ENGINEERING

Convirtiendo datos crudos en vectores de alta dimensionalidad. Los algoritmos de clustering requieren entrada numérica unificada.

ASSEMBLER.PY 
from pyspark.ml.feature import VectorAssembler

assembler = VectorAssembler(
    inputCols=["frecuencia", "monto"],
    outputCol="features"
)
data_with_features = assembler.transform(customer_data)

> MODEL TRAINING

Ejecutando algoritmo K-Means. Iterando centroides para encontrar grupos naturales dentro del espacio de características.

KMEANS_CORE.PY 
from pyspark.ml.clustering import KMeans

kmeans = KMeans(k=3, seed=1, 
                featuresCol="features", 
                predictionCol="cluster_id")
model = kmeans.fit(data_with_features)

> REALIZAR PREDICCIONES

Aplicando la transformación del modelo entrenado a los datos de entrada para etiquetar cada registro.

TRANSFORM.PY 
predictions = model.transform(data_with_features)

> ANÁLISIS DE RESULTADOS

Extracción de insights. Comparando el promedio de consumo y frecuencia por cada clúster identificado.

ANALYTICS.PY
