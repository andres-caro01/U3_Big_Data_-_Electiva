Mostrar datos importantes, representativos y sencillos

Se debe evitar representaciones gráficas de datos irrelevantes o innecesarios o que vuelvan complejo y confuso su representación, como por ejemplo darles efectos 3D o sombras de color, que confunden y lo vuelven más complejo.

Seleccionar el tipo de gráfico adecuado

Lo más simple es utilizar gráficos de barras para comparaciones, gráficos de líneas para representar series de tiempo, diagramas de dispersión para relación entre dos variables y datos geográficos utilizar mapas.

Maximizar la relación datos-tinta

La mayor parte de la "tinta" en un gráfico debe usarse para representar los datos.

Proporcionar contexto

Usar etiquetas claras, títulos informativos y leyendas para que el gráfico sea autoexplicativo.
